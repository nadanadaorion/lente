# Branding de ORI♡N LENTE

Sube a esta misma carpeta los siguientes archivos, respetando exactamente sus nombres:

- `logo.png`: logo principal que reemplaza el círculo verde. Se recomienda PNG transparente, cuadrado, de al menos 512 × 512 px.
- `favicon.ico`: favicon de compatibilidad general. Incluye tamaños 16 × 16 y 32 × 32 px dentro del archivo.
- `favicon-32x32.png`: favicon PNG de 32 × 32 px.
- `favicon-16x16.png`: favicon PNG de 16 × 16 px.
- `apple-touch-icon.png`: icono de 180 × 180 px para iPhone y iPad.

No hace falta editar el HTML. La aplicación ya apunta a estas rutas. Mientras `logo.png` no exista, se mostrará temporalmente la “O” verde anterior.
